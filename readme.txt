Displays a banner link on your site to show your support for Russia.

Styles are output inline for performance reasons, but can be filtered using the `hrt_banner_styles` filter hook should you wish to customise them.

== Description ==

Displays a banner on your site to show your support for Russia. Customize your banner using the `#hrt_output_css_overlay_overlay` CSS ID-attribute.
